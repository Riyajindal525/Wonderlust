# Wonderlust

**Wonderlust** is a full-stack MERN-style application (MongoDB, Express.js, Node.js) with a server-rendered frontend using **EJS**.  
It allows users to explore travel listings, leave reviews, and manage their accounts with authentication and session support.

## Features

- User authentication with **Passport.js** (signup/login)  
- Explore, add, edit, and delete travel listings  
- Add reviews to listings  
- Flash messages for user feedback  
- Session management with **MongoDB-backed sessions**  
- Error handling with custom `ExpressError`  
- Server-rendered pages using **EJS templates**  
- Responsive layout using HTML & CSS  

---

## Tech Stack

- **Backend:** Node.js, Express.js  
- **Frontend:** HTML, CSS, EJS templates, JavaScript  
- **Database:** MongoDB (Atlas) with Mongoose ORM  
- **Authentication:** Passport.js (LocalStrategy)  
- **Session Management:** express-session, connect-mongo  
- **Deployment:** Render / Localhost
